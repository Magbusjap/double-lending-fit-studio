# Fit Studio — Лендинг фитнес-клуба

Учебный проект по вёрстке. Лендинг для фитнес-клуба с новостным блоком, секцией тренеров и блоком FAQ.

## Стек

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

- Методология БЭМ
- Адаптивная вёрстка (Flexbox / Grid)

## Что реализовано

- Hero-секция с CTA-кнопкой
- Блок выбора тренера
- Новостная лента с карточками
- Accordion (FAQ) на чистом CSS
- Шапка с телефоном и акционным баннером
- Подвал с навигацией
